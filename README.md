# nairnavin.github.io

Personal portfolio website for Navin Nair - VP Digital Engineering & AI at Mindsprint.

## Features

- VS Code-inspired dark theme design
- File explorer navigation
- Markdown-style content display
- Responsive design for mobile devices

## Pages

- **home.md** - Professional summary and key highlights
- **background.md** - Work experience, education, and certifications
- **blogs.md** - Links to Medium articles

## Tech Stack

- HTML5
- CSS3 (VS Code dark theme)
- Vanilla JavaScript
- JetBrains Mono font

## Local Development

Simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## Deployment

This site is deployed on GitHub Pages at [nairnavin.github.io](https://nairnavin.github.io)
